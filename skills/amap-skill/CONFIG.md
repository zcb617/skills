# 高德地图技能配置指南

## 概述
这个技能集成了高德地图的多项功能，包括：
- 地点搜索（关键词搜索、附近搜索）
- 路径规划（驾车、步行、骑行）
- 天气查询
- 地理编码（地址转坐标、坐标转地址）

## API密钥配置

### 1. 申请API密钥
1. 访问高德开放平台：https://lbs.amap.com/
2. 注册账号并登录
3. 进入控制台创建新应用
4. 为应用添加Web服务API的KEY

### 2. 配置API密钥
在OpenClaw环境中设置环境变量：
```bash
export AMAP_API_KEY="你的高德API密钥"
```

或者在OpenClaw配置文件中添加：
```json
{
  "env": {
    "AMAP_API_KEY": "你的高德API密钥"
  }
}
```

## 使用方法

### 地点搜索
- `search_places("餐厅", "上海")` - 搜索上海的餐厅
- `search_nearby_places("咖啡厅", "121.473701,31.230416", 2000)` - 搜索指定坐标2公里内的咖啡厅

### 路径规划
- `get_driving_route("121.473701,31.230416", "121.480423,31.236276")` - 获取驾车路线
- `get_walking_route("起点坐标", "终点坐标")` - 获取步行路线
- `get_cycling_route("起点坐标", "终点坐标")` - 获取骑行路线

### 天气查询
- `get_weather("上海")` - 查询上海天气

### 地理编码
- `geocode_address("上海市陆家嘴环路1000号", "上海")` - 地址转坐标
- `reverse_geocode("121.473701,31.230416")` - 坐标转地址

## 注意事项
1. 高德地图API有调用频率限制
2. 某些功能可能需要额外开通权限
3. 请妥善保管API密钥，避免泄露
4. 根据使用量可能产生费用

## 权限要求
- Web服务API权限
- 地图、搜索、导航等相关服务权限
