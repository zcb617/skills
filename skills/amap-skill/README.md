# 高德地图技能 (AMap Skill)

## 描述
高德地图技能提供了一系列基于高德地图API的服务，包括地点搜索、路径规划、天气查询和地理编码功能。这个技能可以帮助用户获取位置相关的信息和服务。

## 功能

### 1. 地点搜索
- **关键词搜索** - 根据关键词搜索地点信息
- **附近搜索** - 搜索指定位置附近的地点

### 2. 路径规划
- **驾车路线** - 提供驾车路径规划
- **步行路线** - 提供步行路径规划  
- **骑行路线** - 提供骑行路径规划

### 3. 天气查询
- **城市天气** - 查询指定城市的天气信息

### 4. 地理编码
- **地址编码** - 将地址转换为经纬度坐标
- **逆地理编码** - 将经纬度坐标转换为详细地址

## 使用场景
- 旅行规划：搜索目的地附近的餐厅、景点、酒店等
- 出行导航：获取最佳路线规划
- 位置查询：将地址转换为坐标或反之
- 天气预报：查询目的地天气情况

## 依赖
- 高德地图API密钥 (AMAP_API_KEY)
- requests 库

## 配置要求
需要在OpenClaw环境中设置高德地图API密钥：
```bash
export AMAP_API_KEY="your_api_key_here"
```

## API限制
请注意高德地图API有调用频率限制，具体限制取决于你的账户类型和套餐。

## 示例用法
- "搜索上海的网红餐厅"
- "查找我附近2公里内的咖啡店" 
- "规划从A地到B地的驾车路线"
- "查询北京今天的天气"
- "将北京市朝阳区国贸大厦转换为坐标"
